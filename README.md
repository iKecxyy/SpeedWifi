# ⚡ Internet Speed Test

Una pagina web simple y limpia para medir de forma estimada la velocidad de internet en tiempo real, mostrando los valores en diferentes unidades como **bps, Kbps, Mbps y Gbps**.

## ✨ Características

- Medición automática al abrir la página
- Interfaz minimalista y moderna
- Botones pequeños y redondos
- Opción de detener y reiniciar la prueba
- Animación suave de los números
- Estimación de velocidad más estable usando varias muestras
- Muestra el resultado final más alto / estable obtenido

## 🛠️ Tecnologías usadas

- HTML
- CSS
- JavaScript
